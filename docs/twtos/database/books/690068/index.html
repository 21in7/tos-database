艾米諾特的故事2

Content

自從傑斯堤得知了拉伊瑪想要利用人類將自己的啟示傳給後代的計畫，並將九名魔君主佈局好之後，第一次發生了由魔族殺害負責守護啟示的守護者的事情。
除了那九名之外，雖然還有很多魔君主收到同樣的命令而行動，但第一次獲得成功的卻只有普利米歐艾米諾特一人。{np}普利米歐艾米諾特在利迪亞夏班的箭殺死露克立斯的瞬間;贏了’他不自覺地想這樣大喊，但他沒有這樣做。
這是他偽裝成人類後，爬到宰相之位壟斷國政下，所獲得的珍貴勝利。
可惜的是他沒有除掉當代守護者中的利迪亞夏班與梅伊班。
但那樣做，他們的名聲過於響亮，實在非常難動手。
如今露克立斯已死，只要找出他守護的啟示就可以了。
至於愚蠢的人類帝王卡杜梅爾，只要幾句話就能讓他把所有的事交給自己負責，打道回府，此處的一切裁示都能按照自己的意思來達成。{np}過去沒有人成功奪取的拉伊瑪的啟示，如此燦爛的功績，都將完全成為他的。
此時利迪亞夏班走了過來。
艾米諾特的人類外貌看起來較她年長，地位也是宰相，但他還是一如既往地意識到周遭的目光，鄭重地向這位受歡迎的女弓箭手禮貌地說[我本就認為拿下叛亂的首腦露克立斯之人，不是軍營裡的任何一位將軍，而是利迪亞，我的料想果然沒錯。]
利迪亞夏班不發一語的望著這位狡猾的政治家，嘆了一口氣後說道[你要是再敢強求我什麼，我絕對不會放過你。不管你在何處，不管你擁有何種爵位，只要我下定決心，你就不會是安全的。]
艾米諾特過去聽到人類說這種話，心情一定會十分不悅。現在的他為了不打破以人類外貌生活的這段時光所維持的平靜，溫柔地說[你也跟你剛剛殺死的師兄一樣，太過自滿了。不管是誰，能爬上我現在這個爵位的，都不是簡簡單單爬上來的。]{np}說著這句話的艾米諾特，和利迪亞一樣，兩人都是話中有話。兩個人對彼此的談話，越來越不像是人類與人類之間的正常對話。
利迪亞夏班開口了[不曉得你聽說了沒有，最近某個蠢蛋講了和你類似的話之後，也死在了我的箭矢之下。]
這句話不是假的，不只是剛剛利迪亞夏班提及的魔族，還有為數不少的高階魔族也不敵利迪亞夏班，或者死亡或者消逝。
雖然艾米諾特覺得那些傢伙根本無法與自己相提並論，但這個女人卻不是那種自己揭示原本的力量後，就能輕易殺死的人。{np}如果問起在這個時代中，哪一個人類能殺死魔王級的魔族的話，只有兩個人可以辦到，而其中一個人絕對就是眼前的利迪亞夏班。
利迪亞的師父梅伊班雖然以聖職者的身份達到了最高境界，但是按照他的個性來看，他只會一邊護著自己，一邊對付魔王，所以絕對無法殺死魔王。{np}艾米諾特並沒有被利迪亞夏班挑釁。
只要他持續維持人類的身份，守著宰相的位子，利迪亞夏班的箭矢再怎麼銳利也與他無關。
{np}然而利迪亞夏班為艾米諾特準備的不是有箭尖的箭矢，而是言語的利劍。[你以為露克立斯師兄死了，你就能得到想要的東西嗎?我想你大概忘了，女神也授予了我同樣的任務，如果露克立死的死可能能讓你得到那個東西，
那我絕對不會親手結束他的生命。不管你的下一步是什麼，我敢肯定，那不會有什麼用。我要離開這裡了，希望你向王好好交代，別讓王責怪我，說那個人竟然連招呼都沒打就離開。]{np}利迪亞夏班說完，就把帽子脫下，對他敬禮後離開。艾米諾特聽到那番話也並沒有太過於擔心，但是過了十天後，他開始感到了絕望。
