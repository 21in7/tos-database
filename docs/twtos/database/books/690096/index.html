*克里戴拉歐泰羅的日誌1
{np}狹小簡陋的房中，躺臥在粗糙床鋪上的男子最後一次伸出手。
克里戴拉握住了那個男子，也就是她父親伸出的手。
{np}[抱歉了，克里戴拉。好歹也是出身貴族，卻死於這種境地，都怪我太無能，沒能為你做些什麼，真是抱歉了。]
{np}克里戴拉歐泰羅握住虛弱的父親的手，搖了搖頭說道。
[別說這種話，請振作起來。]
[若是有你爺爺那時候的財產，還能讓你的未來有所不同的…]
[我的未來我會自己看著辦的。]
[對不起。]
{np}重複著那句話，克里戴拉的父親再也沒有多餘的氣力，閉上眼陷入沉睡。
看著父親，餓了兩餐的克里戴拉也敵不過疲憊與飢餓睡著了。
當她從睡夢中醒過來時，父親已如沉睡般離開人世了。
{np}父親簡陋的葬禮在村民的幫助下好不容易結束，還只是個十幾歲少女的克里戴拉，如今獨自在空房子裡，除了哭什麼也無法做，而當眼淚也終究乾涸時，有人敲響了門。
{np}即使克里戴拉沒有回應，門外的人看門是開著的，又聽到門內有動靜，便推開門走了進來。
進來的兩個人一位是村長，另一位則是個素未謀面的男子。
{np}那個男子確認過克里戴拉的狀況後，鬆了一口氣然後接著說道。
[父親不久前在這種情況下遇到我這種人，又聽到不想聽的事，應該會很辛苦…]
{np}克里戴拉不等他把話說完，抹了抹眼角，故作堅強地說道。
[歐泰羅家門就算在絕望的情況下，也絕不會推遲該做的事，或者迴避責任。]
男子聽了她的話，臉色稍有驚訝然後說道。
{np}[以這個年紀的小女孩來說，你很了不起呢。大家對你們家門的評論中，有人說過沒有繼承人擁有初代馬涅爾歐泰羅的一半資質。]
子邊說邊觀察克里戴拉歐泰羅的表情，即使聽到侮辱家門的言語，她也不為所動。
{np}她並非不懂而是她有自制力。
男子充分了解這點，打從心底再次認可這個女孩。
男子帶著這分認知開始說道。
{np}[第38代國王泰勒馬涅下賜名號及爵位，馬涅爾歐泰羅成為家門始祖後，在傳承到第五代的你過程中，領地早已在三代前就轉手，財產和收藏物也都被賣掉或抵押了。
{np}特別是沃德國王時代的大畫家薩格雷(Sakkurrhe)的畫作，無謂的自尊心導致散盡財產，實在可惜。那些名畫幾乎都已落入他人手中，如今連你父親空有明目的貴族爵位也與你無關了。]
{np} [這些事我很清楚，您不必再說一次。您是誰呢?講這些話。]
[啊！真是的，我忘了自我介紹了，居然會犯這種錯。我是從凱都拉商團來的克迪特(kdlit)。在這種不是太好的情況下見面，雖然不是挺適合說很高興見到你，
{np}但畢竟我們是非見不可的，可是你年紀小小就居然知道我說的事實，還真是了不起啊。無論如何還有件事你必須知道。]
{np}克里戴拉歐泰羅沒有回答，只等著克迪特開口。
[首先，雖然你的父親作為貴族出生然後去世了，不知你知不知情，但我國的貴族地位世襲並非是永久的。歐泰羅家門在始祖死後，一代代對王國均無貢獻，還散盡了家門的名聲與財產。
{np}因此從你這代開始就不是貴族，你不再享有貴族的各種特權，特別是針對稅金的優待。其實這並不成問題，因為你們家並無財產。這個家以及就算只是這裡的一個碟子也好，如今都屬於我們凱都拉商團的。
{np}幸好你父親還算明智，知道要向凱都拉商團貸款，而不是其他地方。我們凱都拉商團並不是那種會為了多減少一點損害，就掏光孤兒少女所有財產的那種人。但是即便如此也不能讓你繼續住在這裡了。
{np}即使不是為了錢，反正這是不被允許的。因此在你成年前，你將會被送到你的監護人那裡。]
[監護人？]{
{np}[不然就要送去聖職者們經營的孤兒院，我想比起孤兒院，監護人還是好些。大概就是被送到自己所屬地區的領主大人那邊，在那裏做些雜活，也能接受教育。為了成人後能自力更生，大多是學些技術，但是若是有天分，任何事都能學的。
{np}根據王國法，不論教育費用多高，地方領主都有義務支付。當然不屬於基本教育水平的昂貴課程並不是免費，等你成人還是必須慢慢償還。]
{np}[那麼我就得去此領主大人的城囉？]
[有點不同…本來是那樣沒錯，但你的情況不太一樣，有別的資助者出現了。]
[是誰？]
[那個無法在這說。即使如此你也不必擔心。監護人並不是說想當或是錢多就能當的，我們凱都拉商團並不是那種只計較錢，對別人的事完全不管的那種團體。]
{np}克迪特停了一下繼續說道。
[我們商團會代替忙於生計的村人，送你去監護人那裡。
{np}這話雖然滑稽，即使是和我們商團負責輸送貨物的人一起順便送你去，但是真要算起來，這可是個虧錢的活呀。
{np}反正這老舊的房子和生財工具壓根還不完你父親的債務，一開始就註定賠錢，所以家裡的任何東西，你想帶都可以帶。
{np}啊！當然在外面藏了什麼東西也一樣，但僅限於你自己背得動的東西。選一個包包，帶上你覺得需要，或者蘊含回憶的東西吧。
{np}就算裡面藏了什麼金塊，我或者商團也不會在意的，建議你盡可能多帶點。當然除了你帶走的東西，留在這裡的所有物品都將歸商團所有，會被活用或者賣掉喔。給你幾個小時的時間，希望我們再次前來時，你已經做好離開的準備了。]
{np}聽完那些話，克里戴拉一表示明白，克迪特跟著就離開了。留下來的村長說了一些村民過世留下年幼女兒時該表達的慰問之後也跟著離開了。
村長走到門外發現凱都拉商團員克迪特並未走遠，就在附近。
{np}為了給克里戴拉一點獨處的空間，兩人一同走得更遠些。等有一定距離後，村長說道。
[先前都不知道凱都拉商團是如此有慈悲心的團體。]
{np}克迪特羞愧地笑著並說道。
[我們商團絕對不會做虧本事的。在某些地方大方施予，某些地方則大肆收割，只是達成平衡而已。往好處想，也可以說是為了在某處大方施予，才在其他地方有效率地累積財富的。]
{np}村長雖然可以輕易反駁克迪特所說的話，但是他沒有刻意說出來，
反而這樣說道。
{np}[話說回來，有資助者出現，對克里戴拉來說真是不幸中的大幸。我們村子的處境不好，如果無法照顧她，就得送她去領主那裡，那麼就得為他人做牛做馬，然後再被送去其他地方，一輩子就這麼過了…]
{np}[那種人生也不算差啊？]
[好歹也曾經是貴族子嗣，我覺得如果有其它路可以走，走其它路都比這個好。]
克迪特沒回話，僅以點頭表示同意。
{np}但其實他心裡這麼想著。
「但是若無法向監護人證明自己的價值，她說不定會覺得不如在這小農村當佃農的妻子都還比較好。但是對現在的克里戴拉來說，她並沒有選擇要過何種人生的機會。不過看她表現出來的意志與態度，未來會如何還不得而知呢…」
