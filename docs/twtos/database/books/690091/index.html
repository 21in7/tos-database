*王陵的傳說
{np}此文件是最近才被發現的資料，目前還無法得知其正確的來源。
文件的用看起來雖然是紙，但又不能說是紙的材質編輯成書，這個方式可以說研究目錄學的學者也相當困惑。
{np}我個人認為這個東西出自於巴伊寶拉女神掌管的幻像圖書館。
{np}這份文件在神樹之日以前除了在世界上沒有記錄，也根本沒有存在過，神樹之日除了我們的世界，對幻想圖書館也造成了影響，我猜這才使文件因此流到我們的世界，或者如果你相信最近的傳聞，這可能是某個啟示者進入幻像圖書館，才導致文件流出。
{np}然而更遺憾的是，這文件的頭尾的內容都被去掉，目前只剩下下方記述的部份了。
女神現在只企盼著有一天能查看這份文件的其他部分。
- 王國曆1096年。
賢者大師魯帕斯 凱赫爾
{np}卡莫夫公爵走在通道上，感受到石造建築物上特有的寒氣。
也許是因為因為大王的命令，所有的工人都被請離，所以更感覺到冷。
但是沒有感受到一絲濕氣，他確信，這是王陵當初的工程十分完善的緣故。
{np}濕氣對於他現在感受到的寒氣雖然會有幫助，但長久來看，這會是對王陵耐久性產生傷害的原因之一。
但是抓不準能支撐多久的建築物，就跟沒有足夠的耐久性是一樣的。
{np}卡莫夫公爵把這份思緒放在腦中，快速移動腳步到他的君主等待的下層。
{np}他的君主，是統一這片土地的君主，是有著大王稱號的人物，本名叫做札卡里耶爾。
當然這個名字在他的記憶中，已經好久沒聽人說過了。
{np}只有見到大王的第一天除外。
君王也會有名字，但當然不會有人直呼他的本名。
{np}札卡里耶爾這個名字，對於大王還是領主時，就當其家臣到現在卡莫夫公爵來說已很陌生。
卡莫夫公爵一走到大王札卡里耶爾與其臣子們停留的地方，大王就欣喜地開口了。
{np}[雖然還沒完工，但我很高興王陵的工程如我所願地進行，你辛苦了，公爵。]
對於君主的讚賞，公爵說出了心裡話。
[陛下，王陵各處都尚未完工，這麼快就請工人走，似乎太早了。]
[又不是讓他們全部都走，不是只有這一層的人離開嗎？]
{np}[這層也還有很多地方尚須完善啊。]
[朕認為現在該做的，是不該隨便將那些內容洩漏出去。]
[可是陛下，刻在峽谷上的文獻雕刻，也動用了許多人啊，我認為現在還是該將他們召回才是。]
{np}[聽從朕的命令，在堅硬的石材上刻上文字與圖案的人們沒有任何罪，但朕還是強制將他們這段記憶消除了，因為我怕魔族或帶有邪惡目的之人會覬覦朕的遺產，所以可能的話，我不想再對百姓做同樣的事了。]
{np}卡莫夫公爵為了讓大王改變心意，想請求幫助，因此他觀察著隨侍在大王身邊的臣子們。
{np}大王身邊有服侍最久的老臣諾培蓋勒，掌管所有兵力的總司令官約彼特奇連，以及負責護衛葛坎多尼修將軍等等高官，但卻沒有人附議卡莫夫公爵的話，站出來發聲。
{np}反而是公爵戴農夏瑞比開口了。
[若是陛下親自動手，為人臣的，哪有旁觀的道理？卡莫夫公爵您負責掌管王國的技術與物產，一切講求利害與效率，但我認為您應該將眼光放長遠一些才是。]
卡莫夫公爵立刻就接話了。
[我並不是不想親自為陛下效力才說這些的，只是既然已負責掌管國政，那便有更重要的事要做，而不是在這裡做著石匠的工作。]
{np}戴農夏瑞比沒有因此住口。
[我們都已經聽陛下說過了，雖然我不清楚內情，但陛下建造這處王陵，並不是出於他個人的私慾。
{np}這是為了應對將來王國面臨的災禍，甚至是可能會左右所有人類未來的預言，這些事如果不由我們來做，那該誰來？]
{np}掌管王國刑法與司法的最高階貴族說了這番話之後，擔任大王護衛的葛坎多尼修這才開口。
{np}[二位說的都很有道理，因此下官請求陛下，這件事就讓下官帶領的王國騎士團來為陛下效勞吧。親衛隊中，絕不會有任何人將這裡的事洩漏出去，而且今日下官帶來的，都是親衛隊中尤其值得信賴之人。]
{np}此話一出，身為宰相的老臣諾培蓋勒接話了。
[我知道陛下怕留在王陵中諭示秘密會洩漏出去，但卡莫夫公爵的建言在先，況且在場的既是最高階的貴族，同時也是大王的忠臣，如果這些人都不能相信，實在太說不過去了。
{np}所以必須找個不對石匠與工人造成負擔，同時也能讓我們放心的方法才行。]
[有那種方法嗎？]
札卡里耶爾一問，諾培蓋勒再說道。
{np}[老臣認為，即使陛下和臣子們親自動手，為了國政著想，時間上也要越短越好。因此...]
[因此？]
札卡里耶爾再問。
{np}[…第一，陛下必須先同意使用魔法。第二，捨棄用槌子或鑿子，用能溶解岩石的強酸或藥品，將事先寫在岩石上的字溶出形狀來。]
[魔法與藥品，這兩種東西不會很危險嗎？]
{np}這位年邁的忠臣，除了賢明，也因為已侍奉大王很久的關係，他像是聽懂了札卡里耶爾這段簡短的話一般，很快地就回答了他的疑問。
[藥品的部分我們現在好好處理就沒問題了，另外魔法的話，我認為這不會對未來來到此處的人造成問題。
{np}陛下正是擔心，用魔法留下的諭示，日後會帶給哪個厲害的魔法師線索，這樣一來您想保留的傳承，怕是會落到不適切的人手上呀。
{np}不管是魔法的火焰造成的印記，還是一般的火焰造成的印記，時間一久，都只是一般的印記罷了。
{np}牆面與碑石上的文字的圖樣也會用這種方式，您無須擔心刻的過程中使用的魔法。]
札卡里耶爾大概是很滿意這個回答，接著說道。
[嗯…這樣就行了嗎？]
{np}[如多尼修將軍所說，若使用他指揮的將士們，其它的大臣可以更專心在他們的本職。而峽谷的作業已經全數完成，卡莫夫公爵的報告，依我判斷，此處王陵的工程只剩一點收尾的部分而已，各方面來看都綽綽有餘。]
[我知道了，就按照傾的意思辦吧。]
{np}[非常謝謝您，陛下，但下官還有一事不解。]
[何事？]
[依老臣來看，陛下對王陵的設計，似乎不是為了陛下的長眠，而是考慮到後代來此處之人而構想的。
{np}因此看起來，到目前為止大部分的工程都是為了試驗該人的資質與能力所做的安排。
而且外面都是為了抵禦敵人與盜挖的裝置，下官疑惑的是，這與我們現在要做的事似乎無關。]
{np}[正如卿所言，目前為止完工的部分都是為了試驗而做的。但其實在朕的心裡，會做這些，並不全是只為了考驗那位追尋女神啟示而來的人。
{np}其實朕想給那位通過考驗之人禮物，因為朕雖無法親自考驗他，但他就算沒有朕的考驗，勢必也得走上艱辛的道路。所以朕想在朕長眠的地方留下禮物給他，另一方面，朕也有想告訴他的教誨。
{np}說不定到時候這些教誨已經過時，但朕也不打算因為這樣就放棄。]
聽了這番話後，諾培蓋勒微笑地對札卡里耶爾大王說道。
{np}[這是陛下與建立這個國家的建國者們親手寫下的教誨，絕不可能沒有用的，不如將王陵的此處取一個與教誨相關的名字如何？]
{np}[教誨的殿堂，你是說這樣命名嗎？雖然有點難為情，不過也不壞。]
宰相諾培蓋勒再次回答。
[這個王陵必須等多久的時間才能迎來期盼的人，實在是難以預料。
{np}即使有一天這些都沒用了，但還是盡可能地留下越多越好。
畢竟我們不知道在那個時代，我們所有的知識是否都已經過時。]
{np}[我也正有此意，由其是朕所知道的關於女神的神學，以及對魔族的知識、這個王國相關的建國歷史、女神的指導對朕的設計與建築造成的影響、說不定在那個時代能派得上用場的技術與魔法師的咒文等等。
{np}可以的話，朕想將所有的東西都留在這個王陵，並且在這王陵還有空白之處的時候，將那些秘錄都記載上去。]
{np}說著此番話的大王眼神充滿著堅定，而且從他臉上看得出這並非純粹是一個人類的強烈意志，而是從創世開始到現今的‧女神的其中一道意旨，在場的臣子們都不由自主地低下了頭表示敬意。
{np}附註 : 最近據博學多聞的學者希瑞尼亞歐德說，王陵中並不存在與研判與上述內容有關的大王遺產。
對此番見解進行調查後，有了如下的可能性。
{np}第一個可能是，此文件並非事實。
但我直覺地排除這種可能。
第二個可能是，隨著王陵開放，發生的事故與汙染造成札卡里耶爾大王的遺產被汙染或毀損。
{np}第三個可能，是被視為可能性最高的，便是傳說中在大森林地帶某處的，有著神秘的王陵，裡面有著大王真正的遺產。
也就是說，因為某種原因，大王無法將那些秘密遺產放在自己的王陵，而是在後代的國王繼承後，藏在了其他王陵。
{np}若真是如此，到底是因為何種原因，讓有緣為其後代之人不選擇大王本人的陵墓，而是讓遺產放在其他陵墓，以目前來說，並無法得知
{np}但前提是，在還無法確定大森林地帶是否真有王陵的情況下，我們只能承認，大王與其後世的繼承人是因為何種緣故下了這種決定，已經無法考據。
{np} - 王國曆1096年。
卡歐尼拉公主執政元年
