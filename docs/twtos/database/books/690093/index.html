關於普魯達

[我們的想法一直都沒錯，像巴烏巴斯你這種人，才是照自己的意思解讀創造神的遺志，並且獨斷行動，尤其是吉特涅最為嚴重。]

{np}針對羅迦娜嚴肅說的這番話，奧斯特雅雖短暫接觸到他的眼神，但邏輯上無法反駁他，若是說得太多，最後會變成必須聲討吉特涅，因此他還是別過頭，避開了他的眼神。

[我們不是一起創造了世界，並共同擔負起對這個世界的責任嗎？雖然我們彼此有一些意見不合，但也能僅止於言論上，這樣我們才能為了這個世界做事。重點是，我們在等待的傢伙，似乎已經到了。]

[狂妄的東西。]
{np}語畢，巴烏巴斯立刻有了動作。

普魯達自認能躲過攻擊，卻反而中招。
不同於對付其他的龍，這次他的速度更快，攻擊也更凌厲。
{np}普魯達側面感覺到了前所未有的疼痛，他用翅膀上的爪子勉強擋住了第二次的攻擊。
不對，本以為是擋住了。
然而在那瞬間，他的爪子應聲碎裂。
{np}與腰部的傷不同，爪子雖然能在短時間內恢復，但打破普魯達防禦的巴烏巴斯，又再度把手瞄準了側面。
在爪子碎裂的情況下，其實可以攻擊像翅膀這種更近的部位，但巴烏巴斯又瞄準了龍的身軀，就代表他並沒有使出全力。
{np}[看來不需要用上我的權能，也不用拿出武器了...]
普魯達感到很驚慌，他發現自己一個人，並不是巴烏巴斯的對手。
{np}若巴烏巴斯沒有其他同伴協助，而自己的同伴也能平安無事與自己並肩作戰，也許還有機會勝利。
自己與同伴們，長久以來在原來的世界過著很滿意的生活。
{np}在發現其他的世界後，由於過度自信，而不想了解此處所謂的神。
然而就因為這種事不關己，導致需付出的代價未免太大。
不僅被滅族，自己也命在旦夕。
{np}普魯達揮舞右側的翅膀，成功將巴烏巴斯推開，接著進行連續的攻擊，試圖要咬巴烏巴斯，成功將他驅趕到更遠的地方。
然後他用力驅動翅膀，試圖升空。
他認為比起直接逃跑，在上方攻擊會比較有利。
{np}就因為如此，在他發現自己飛不起來的時候，簡直驚訝不已。
普魯達依舊無法了解巴烏巴斯的表情，而巴烏巴斯露出了滿意的微笑，抓住了普魯達手足無措的瞬間。
這次他往普魯達的頭部進行了攻擊。
{np}此時的普魯達，側面與頭部的傷勢帶給他前所未有的痛苦。
過去從來沒有人給過他如此重創。
當然，在原來的世界中也曾發生過同族間的爭鬥，有些龍受過比這個還重的傷。
{np}其中有些傢伙受不了疼痛，大聲哀嚎，也有些傢伙硬是咬牙忍耐著。
而普魯達一直都是勝利的一方，沒有感受過傷口帶來的疼痛。
然而他現在從一個陌生世界的陌生生物身上領悟到了，無論他身軀多麼強壯，其實承受疼痛的能力並沒有那麼強。
{np}從另一方面來看，也能說是他運氣不好吧。
總之靜下心想，他還是沒有可能贏。
所以他必須逃跑。
但是已經無法飛行，只能拖著受傷的身體，在地面上移動，擺脫巴烏巴斯。
{np}巴烏巴斯悠然自得地做出了一個殘忍的微笑，緩緩地跟了上去。
普魯達雖然努力跑著，但無論如何還是無法超前此地的魔神巴烏巴斯的速度太多。
普魯達的負傷與巴烏巴斯的權能，形成了一個奇怪的追擊狀況。
{np}此時奧斯特雅靜靜地來到了巴烏巴斯的身邊。
巴烏巴斯在他還沒站到自己的身邊之前，光是靠著翅膀的聲音，就已經察覺到是奧斯特雅。
巴烏巴斯一回頭，奧斯特雅便說道。
{np}[放過他吧。]
[呵呵，真是可笑，難得你說出這麼溫柔的話，真是讓我不得不問了。怎麼？為什麼要放過他？]
[因為這也許是僅存的最後一隻龍了...]
{np}[話雖如此，但他們原本的世界有可能還有倖存者，其他女神前去的地方也說不定也還有。]
[但至少這裡已經沒有了。]
[要是放過他，說不定會對人類造成危害！那傢伙分不清人類和我們的差別，要是普通人類，也許就直接被吃掉了！]
{np}[放過他，也許會躲到地底的某處吧。如此一來，擁有大地權能的澤米娜一定會知道，到時候再將他封印就行了。一旦被封印在地底深處，若沒有澤米娜甚至是創造神的力量，是沒人能解封的，就算是奧施莉奈或吉特涅也很難做到。]
{np}[還真像你軟弱的作風，好吧，難得你拜託我，我就照辦吧。但我想知道兩件事。]
[你想知道什麼？]
[不，是三件事才對。第一，什麼時候才能再聽到你說這麼溫柔的話。第二，什麼時候才能看到奧斯特雅拼命的樣子。最後一項，什麼時候普魯達會被解封。]
{np}[三件事都不可能。]
奧斯特雅的表情再次變得嚴肅。
[呵呵，話別說得太死，說不定你有一天還會再求我，說不定終有一天我們會兵戎相向，而創造神也不會再甦醒，或阻止那個力量再次侵襲世界。]
{np}[你是想要放過他的代價嗎...？]
[現在就給我吧，我雖然喜歡殺生，但我不喜歡善後，這些四散的屍體，總是要有個人來清理，我的份就拜託你了。]
巴烏巴斯說完這番話，不等奧斯特雅回答便離開了。
{np}奧斯特雅比起巴烏巴斯丟給自己的清理工作，對他剛剛說的話，產生了不好的預感。
「要是發生什麼事，拉伊瑪應該會告知吧？」
他這麼告訴自己，接著對眼前這許多的龍屍與殘骸嘆了一口氣後，開始了清理作業。
