蘇科姆的星之塔攻略紀錄6

Content

「若是要靠力量，應該要在塔完工之前執行才對」蘇科姆想著。有這樣想法的蘇科姆，只能選擇發動總攻擊了。想要雇用人類傭兵，導致他受到魔界上層的強烈批判之後，他需要使用更強硬的策略才行。而即便這不是最好的辦法，但也是必須嘗試的方法之一。{np}雖然覺得不可能，但是還是希望星之塔的夏班史達放鬆警戒心，部下們不要因為沒用的挑釁而犧牲，持續地召募兵力，驅使弗貝塔將魔族們聚集於魔界或是透過徵兵增加兵力。
魔界有傳聞說蘇科姆被魔王們責罵，弗貝塔經常不滿地透露籌措兵力和支援的事很困難，
即便如此還是盡全力將戰力提升，而且蘇科姆這次也下定決心要走到最後。{np}就這樣花費了半年的時間準備著總攻擊的事情。可飛行的魔族缺很多，船隻也很難取得，只能放棄在水上的攻擊，所以大部分都只能從陸地進攻，但這也是沒有辦法的事。{np}這個湖水的漂浮石正好在絕妙的高度漂浮著，所以只到腳踝高度的湖水，對魔族給予了足夠的傷害，而那樣的深度又無法讓船隻行駛，所以又阻斷了使用船隻的方法。
利迪亞夏班利用魔法漂浮石，讓湖水的水位永遠維持在腳踝的高度。雖然夏班史達團員們因鞋子永遠會濕搭搭而感到有點不方便，但對魔族來說，這可是致命的問題。{np} 沒有戰鬥或戰鬥訓練時，夏班史達的實習團員們會帶著像爪子般的工具，去把地板上的青苔與水草割掉。
他們會小心翼翼的走到魔法漂浮石的角落位置，然後將收集的草與水草丟到湖水之中。
這時會有許多湖水的水生動物來吃那些草，後來甚至會為了隨時吃到不知何時會斷糧的草，直接常態的聚集在角落。{np} 夏班史達的新團員也因為這樣，必須熟知到哪裡為止的魔法漂浮石是可以走的，從哪裡開始是無止盡的湖水深淵處的正上方。
而實習期間會透過水草作業來更加確定那些位置。當然一旁還是會有老練的前輩們為了防止新團員失誤掉入水中而在一旁監督著。{np}從邊緣掉落水裡的事故的危險度超過想像，
因為如同我說過的，湖水中的水生動物們會為了吃荒廢的水草而聚集在一起。
他們踩過邊緣時還不怎麼危險，但是只要有人掉到水裡，就會沒理由地攻擊，要從這攻擊之下存活是一件非常難的事。
這樣的攻擊讓在水上攻擊星之塔一事變得困難也是理由。
因為這些生物，水上攻擊時，如果船被破壞，就算不是魔族會游泳，也沒有可以上岸的地方。
就算很會游泳、體力好，可以到達邊緣地帶也無法存活，因為邊緣是最危險的地方。{np}因此沒有魔法漂浮石的水域，比周圍有魔法漂浮石的位置更加安全。
但那前提是有足夠的體力。
被夏班史達團員保護的居民們也沒有受到這方面的威脅，這不是因為凶暴的生物被夏班史達餵食的水草餵食的關係。
剛開始他們都是肉食性的生物，是利迪亞夏班在進行嚴密的長期作戰計畫時，依照計畫讓他們中了必須靠水草維生的毒。{np}因此漁民們在沒有兇暴魚類的中心水域捕魚，而那裡因為少了肉食性的魚類，也讓很多漁獲大量繁殖，使他們能夠豐收。
雖然飢餓的肉食性生物有時還是會回到那裡抓一些小魚來吃，但等到填飽肚子時，還是會回到角落的位置等著水草。
雖然偶爾會遇見漁民們，但漁民們身上都會有緊急時可以使用的水草乾，危急時可以把水草乾拋的遠遠的，然後趁機逃離，所以完全沒有任何漁民陷入過危險當中。{np}只是夏班史達團員們為了不讓水中的怪物們看到漁民就聯想起水草而對他們發動攻擊，所以還是盡可能的制止了這種餵食的行為，漁民們也嚴謹的遵守著管制，只有真正緊急的時候才會使用。
不過，水草都會固定在角落的位置大量出現，所以那些水中怪物幾乎沒有因此而更改習性。{np}所以人們會走動的湖水，一直都維持著乾淨的狀態。
透過那些乾淨的地板，蘇科姆的部下一邊丟著石頭，一邊踩在那上面慢慢的往星之塔靠近著。{np}倒入大量泥土的同時，一步一步的向前走去。
籠城戰的基本雖然是將圍繞於城周圍的護城河填平，但現在整個湖水都是護城河，歷史上也沒有將整個湖水都填平的這種荒唐類似案例。
幸好水的高度只有到腳踝而已，造出一步的路徑應該是不會很困難，只是抵達塔之前要跨過的湖水距離和範圍實在是太遠且太寬廣了。
多數的兵力都分散開來，只要不要進攻的話，就不會成為星之塔投射武器的目標了。{np}充滿氣勢的下達了攻擊命令，卻沒有發生任何的戰鬥，整整一天就這樣過去了。
這件事情雖然很奇怪，但也是理所當然的，因為魔族忙著建造立足於湖水的地方，而且這裡也還在星之塔投射武器的射程之外。
就算是利迪亞夏班也無法限定星之塔內可設置的弩炮數量和射程距離。{np}由魔法漂浮石所組成的區域離星之塔有段距離，所以在遠程的攻擊之下非常安全。
夏班史達只需在塔中設置監視兵觀察工程的狀況，即使休息一下也無所謂。自古以來，當敵人要將城外的護城河填平的話，城裡就會對攻城軍發出各種的攻擊，這種傳統的戰鬥方式已經消失了。{np}直到攻擊的第二天，才開始了那種的傳統攻擊。
因為蘇科姆部下們的工程進度已經進入到射程範圍內了。
夏班史達團員們看著位於湖水中心的星之塔，無法相信這是人類所製造擁有恐怖射程距離的弩炮，朝向魔族的頭上攻擊。蘇科姆的部下們無法做出任何的反擊，只能一個個的倒在地上。{np}雖然有預先準備位於屋頂的攻城塔，然後在裡面進行操作，盡可能的減少損失，但這種裝備在持續的攻擊下也是會遭受破壞的。
如果要說值得欣慰的地方，就是夏班史達團員們為了將這些裝備破壞，利用投石器所丟的石頭，至少還可以用在工程上面。
因為星之塔是建造於水的上方，因此在其它攻城戰可以使用的眾多選項都消失了。
舉例來說，傳統攻城戰所使用的挖坑，就算你很會挖洞，在這裡也是沒用的。{np}當然蘇科姆也很清楚的知道，利迪亞夏班在建造星之塔的時候，不可能不會考慮到籠城戰而做好基本準備的。
如果敵人是湖水的話，那自己就要像大海一樣，就算是動員所有的魔族和怪物也一定要勝利。最重要的是因為魔界的上級們一直在施加壓力的關係。{np}可是填滿淺水的話，進攻的速度就會慢下來，傷亡也只會隨著增加。
依照魔界傾向，就算是傷亡增加，如果是喜歡戰鬥的魔族或凶暴的怪物也不會有所不滿。
但是這裡的情況卻不同，甚麼事情都沒做，傷亡卻一直增加，完全沒有痛快的戰鬥過，下級魔族們的不滿頓時爆發出來了。
如果是跟真正的強敵戰鬥後而死去，依照魔族的傾向，一般情況下都還可以控制得住。{np}可是連敵人的長相都沒看過，除了敵人投射的武器、投石和弓箭之外看不到任何的東西，在這種情況下，下級們對蘇科姆的不滿也越來越大了。
弗貝塔雖然沒說話，但是看著蘇科姆的寶座，似乎不斷地在累積不滿的情緒。{np}隨著時間的流逝，攻擊先鋒已經快到星之塔的入口了。
飛行魔族們也正式加入了戰鬥中，將到目前為止飛射過來的投石在空中攔截後改變方向，減少魔族的損傷。
將一部分的人留下繼續填滿工程，然後對眾多的魔族下達了剩下的距離哪怕是受到傷害也要踩著水往前突擊的命令，發動攻擊後不到幾天，才真正迎來了要發動總攻擊的時刻。{np}當然從星之塔中，往下施展了水、投石、弓箭、火焰等各種的魔法。
從空中進攻的魔族們都被夏班史達團員們的弓箭射中後掉入湖中，就算是掉落在漂浮石上，如果那邊不是隨著進攻而填滿的地方的話，也會遭受到湖水的傷害後死亡。
雖然戰況完全沒有任何的優勢，但蘇科姆一方總算是第一次感受到有在戰鬥了。{np}不管這場攻城戰是勝利還是全滅，蘇科姆都決定孤注一擲了。全部的兵力已經進入到湖水的水域中，自己也站在了前鋒上。
沒有堅不可摧的城，這是籠城戰中永不改變的事實。
如果沒有其他因素的話，一般來說籠城者最後會因為缺乏物資而失敗。
籠城者的勝利需要救援兵在適當的時機抵達或是因為外部有我軍援方而一起攻打包圍者，持續的拉鋸戰後取得勝利。堅固的防禦設備只有爭取時間的功能而已，而輸掉了時間的一方也就代表失敗了。{np}蘇科姆這樣相信著，督促自己戰鬥。
突然蘇科姆有一股不祥的預感，於是看了上方，發現利迪亞夏班正對著自己拉著弓在星之塔的瞭望台欄杆上。
過了很久的時間之後，蘇科姆像是感受到了什麼，望向她時感覺就像是等待她一樣。{np}蘇科姆馬上就發現到了她手中的弓和弓箭正是殺死魔王集布理納斯的武器。
一發就可以將自己生命結束的弓箭。
蘇科姆也知道站在先鋒是一件很危險的事情，因為利迪亞夏班可以狙擊自己。
但蘇科姆判斷，如果不這麼做的話，就無法抑制住部下到上級魔族們的滿腔憤怒了。{np}被傑斯堤叫去還可以忍受，但如果傑斯堤將自己送回給卡魯塔斯的話，為了追究失敗的責任，不知道卡魯塔斯會對蘇科姆做出什麼事情。
成為卡魯塔斯的玩具受到拷問，還不如死在利迪亞夏班的弓箭上。{np}儘管蘇科姆注視著利迪亞夏班，她也沒有射出弓箭。反而將弓箭收起來後，這樣說道[白癡，你們踩著的是用魔法浮在水面的石材結構物。]
儘管距離很遠，對魔君主蘇科姆來說，還是聽得很清楚。
但卻不明白其意義，不過也不需要說明，因為下個瞬間馬上就用身體感受到了。{np}從腳傳來一股震動後，接著整個地板開始往下沉。
一般事情都無動於衷的魔族，也有人開始在大聲尖叫。
從四面八方湧進來的水，開始將魔族燃燒。
看著在水中被火燒死的部下們，蘇科姆卻沒有可以做的事情。
直到這個時候才知道對方一直都在等待足夠的敵軍到來。
別說是攻擊了，連逃脫的距離，也不能用時或分單位而是要以日的單位來計算。
就算擁有飛行或漂浮的能力，也要受到弓箭的攻擊才可以逃脫。{np}和少數的魔族好不容易逃脫出來時，弗貝塔領導著從魔界帶來的魔族們也剛好抵達了。
接著代替失魂落魄的蘇科姆整頓安撫生存者們，讓魔族們重新振作起來。{np}根據弗貝塔之後的調查報告，關於魔法漂浮石上下移動或改變位置的事情，對於星之塔來說負擔也是很大，所以沒辦法經常使用。
根據魔法專家們的見解，使用一次之後，要經過幾十年的魔力充電後才可以再次使用。{np}但當時的蘇科姆或弗貝塔都沒有多餘的時間考慮到這個問題，因為漂浮石的移動使得之前進軍所做的工程都變得毫無用處的關係。
夏班史達團員們從塔中出來後開始整理那些沒有被水沖走而殘留的東西，對於投石有用的就搬到塔裡，剩下的就扔到湖裡了。{np}為了妨礙那個工作也派去了很多部下，基本上戰鬥的同時要避免接觸到湖水是個很大的負擔，從看到一切情況的塔頂發出警報的話，作業中的夏班史達團員們就會分為兩組，一組往塔跑的時候，另一組就用弓箭來攻擊，以這種輪流攻擊的方式慢慢撤退。
蘇科姆雖然覺得很委屈，但因為這場敗戰，在魔族社會中從上到下都失去了聲望。
下級魔族們雖然不敢拿出來討論，但蘇科姆也是都知道的。
只是現在也想不出來如何恢復聲望。
