拉伊瑪的千年傳 第1卷

Content

睜開雙眼的拉伊瑪重整了一下心情。
整理好心情後，被創造神賦予的神之身軀也開始穩定了下來。
透過觀察尚未製造的命運的線索來看待未來並不容易。那樣的未來並不是一定的，因此它是隨時可被改變的脆弱的東西，但又是一定會來到的未來，等已經到了眼前時，往往會到了無法逃避的地步。{np}即使身為女神，在創造神所設定的宇宙法則之下，也不全然的自由。反而大部分的女神都是那法則的守護者，所以更是從屬在那其中。
但是拉伊瑪認為還有自己可以做的事情，所以趕忙在做整理。根據自己看到的預知，雖然時間很充裕，但是因為對手相當棘手，所以不能因為看得到未來而感到放心。{np}從懸涯上掉下來的人類也知道未來。只是他沒有逃避的方法而已。
但是如果因為這樣而失去沈著，那將會後悔一千年，因此拉伊瑪努力的從趕忙與細心中找出一個平衡。{np}接著，認為已經做好了離開自己住所準備的拉伊瑪，重新在看了一遍，這個經歷了漫長歲月與比世界上任何地方都還要有感情的住所。
如果自己預測的未來沒錯，而且預知也必須要正確，大概有千年的時間不會再回到這裡了。
因此想要再一次的把這個即將成為千年懷念的場所收在眼裡，雖然眼神中顯露出深邃的抒情，但拉伊瑪心中清楚的知道，想要努力維持這些記憶的本身，是不合理的事實。{np}如果一切都照著計劃進行，那麼也不會有讓思念堆積一千年的事情發生，而且也一定要這樣才行。
拉伊瑪深深嘆了一口氣，舉起手對四面的牆上用光束寫下了留言。
只會對必要的人顯示的留言，分別寫在個別的牆面上，牆面很快的失去了光芒，恢復成普通的牆面。寫完對不同的人留下的話，已經沒有繼續留在此處要做的事了。{np}根據預知，她離開之後首先到達這裡的原因，是為了關係即將分裂的姐妹吉特涅。
對吉特涅也有一些依戀，因此把留給她的話隱藏了起來，等她來到這個房間時再顯示給她看，但吉特涅是否會尊重這留言還是個疑問。
對於即使這樣也因為留有依戀而留下留言的自己感到同情，拉伊瑪離開了自己的住處，踏上了千年的旅程。
