蘇科姆的星之塔攻略紀錄2

Content

在一開始作戰的那天，看到遠處夏班史達的團員們站在湖水上，用爪子和其他工具挖地板的模樣後，就向一部分部下魔族下達了突擊的命令，
但當先發部隊都還沒碰到水時，星之塔就發現了敵人，並且發出了警報，
因塔的位置很高，負責監視任務的人也都繃緊神經警戒著，所以根本無法突襲，雖然也可以挑有霧的日子，但今日的戰鬥也含有預備測試的含義，除此之外，這湖水會起霧的日子並不多，這也就是利迪亞夏班選擇這裡的原因之一。{np}站在湖水上工作的團員們在聽到警報後，紛紛帶著裝備準備回歸，這時湖水的高度在團員們的腳踝左右，如從遠處看，他們就像是站在水上一樣，總之團員們都想要盡快回到塔那邊去，團員們的動作雖然都很迅速，但卻不見慌張的氣色，
因警報響的太緊急了，所以要找時間點攻擊回歸塔的人看來是不可能的，因蘇科姆沒想到這點，所以他也沒向先發部隊下達催促的命令，但就在先發部隊碰到水時，卻發生了完全意想不到的事情。 {np}就像是掉入了腐蝕液體裡一樣，大家都驚慌失措了起來。症狀也各自不同，有的傢伙被灼傷、有的傢伙被融化、有的傢伙被石化等等。
蘇科姆小聲的自言自語說[不是一般的水..]，但這是在發動第一次攻擊之前，不會知道的事實。星之塔位於可以瞭望整個湖水的位置中，高度也非常足夠，最上層有設置望遠鏡。{np}即使是派出少數的偵察兵，也很難不被發現。因此為了往後的戰鬥，所有的軍隊都必須進行的事前偵測，根本就成了一去不回的軍事行動。{np}蘇科姆對先發部隊下達了撤退的命令，並且命令他們把湖水取來作為樣本，然後輕易的放棄了第一次的攻擊。這時的蘇科姆還有著樂觀的期待。
