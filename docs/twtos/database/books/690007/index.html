擴充自我{/} {np}
恭喜你得到平凡人看不見的書。
雖然魔力的運用方式很多，但鎖鏈法師的方式可說是非常獨特。

其他派別的魔法師會對魔力行使影響力。
也就是說用火球或閃電攻擊敵人，其實並不是利用魔力與敵人連結的行為。
{np}在上述攻擊中，實際的行為是利用魔力操控火與電。
而魔法師所做的行為，是把可控制的能量的方向或終點朝向自己所設定之處送出。
{np}換句話說，鎖鏈法師以外的其他派別的魔法師並不是直接控制敵人，而是藉由魔法來控制自然界的能量，再將其指定到敵人身上，他們只不過是在短時間內進行此過程罷了。
但如果你想踏上鎖鏈法師之路，其魔力的運用方式是完全不同的。
鎖鏈法師使用魔力的方式是，在短間內找出對手與自己的魔力的連接點。
{np}另外，設定兩個以上的目標時，以及這兩個以上的目標具有不同的性質時，鎖鏈法師的魔力運用方式就會變得更加複雜。因為此時鎖鏈法師必須利用自己的魔力，把相同性質的魔力連接在性質互異的兩個攻擊目標的連接點上。
{np}#VALUE!{np}因此必須牢記，鎖鏈法師不同於其他派別的魔法師，必須將其作為確立魔法的自我擴充的道具來使用。
尤其是以後當你與其他鎖鏈法師一較高低時，這將會是更重要的課題。
{np}最後再次恭喜你得到這本書。因為得到這本書代表你成功找到有形世界與無形世界之間的連接點。
因此，如果將來你在屬於自己的領域的目標之間尋找魔力的連接點時經歷困難，那就是你不夠真誠所致。
