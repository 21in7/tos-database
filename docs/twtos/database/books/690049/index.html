烏斯達斯的日誌

Content

X月 13日
-
艾勒瑪.. 艾勒瑪死了..
我暫時離開崗位時，狼群之類的士兵攻擊了艾勒瑪。
看到艾勒瑪那變得變得冰冷的身體之後，我振作起來親手把士兵全部殲滅。
{np}好空虛..
我連心愛的人都保護不了，任務對我來說有什麼用..
我是不是也該隨艾勒瑪而去呢..

{np}X月 14日
-
身穿黑衣的魔族出現在我眼前。
既然已經下定決心隨艾勒瑪而去，還有什麼好怕的。
魔族邪惡地笑者問我想不想救活艾勒瑪。
{np}他手上提著有如克魯比那的血般鮮紅的石頭碎片。
由於此處是雪原，屍體不會腐爛.. 也許我的力量能夠救活艾勒瑪。

{np}X月18日
-
使用克魯比那後，我擁有的魔力馬上大幅增強。
問題是克魯比那就像貪圖我的生命似的，一直消耗
我的生命。
看來應該要更謹慎地使用克魯比那。

{np}X月 X日
-
我感覺到克魯比那漸漸侵蝕我的身體。
就像杯底破裂的玻璃杯般，不使用克魯比那時，我的生命力似乎也在流失。
但事到如今已經不能停下來了。
我已經打破禁忌開始以人類為對象進行實驗。
就算我會喪命，只要能再次讓你活過來就值得了..
我的性命不管怎樣都無所謂..
