艾米諾特的故事3

Content

在露克立斯建造的大地中掃蕩四周的艾米諾特，跪在了降臨於此處的傑斯堤面前。
看著呈現跪姿的艾米諾特，傑斯堤說出了讓人不快的言語。[你借用人類的肉身幹那種勾當幾十年，結果竟然只是殺了一個叫露克立斯的人類，而他手中的啟示你不但沒找到，甚至查不出那個啟示現在在這個要塞的何處?]
艾米諾特觀察傑斯堤的情緒，小聲的說[非常抱歉。]
然而他心裡卻產生了小小的不滿，倘若這個人不是傑斯堤，而是巴以卡的話，或許能了解他的想法。
不僅如此，傑斯堤還接著喝斥[所以你打算怎麼辦?你坐到了這個四不像的人類宰相爵位，最終還是得返回王都，如此一來不就無法繼續探索大地要塞了嗎?
也許你覺得你化成人類的樣子做出了一些成果，可是你看你現在的樣子，一事無成，還被你自己做出來的宰相爵位絆住了手腳，妨礙了你原本的任務不是嗎?]
{np}艾米諾特無法乖乖的聽傑斯堤的質疑，忍不住回答道。[隱藏著真正的目的，擁有可以讓我利用的人類士兵與部下是我擁有的人類社會的權力。所以即使我回到王城，受我命的官吏與士兵們還是會繼續替我搜索的...]{np}傑斯堤被這番話激怒[你都這個樣子了，還想依賴人類?你這傢伙找不出來的東西，那些螻蟻能找得到嗎?你腦子裡到底在想什麼?]
艾米諾特發覺傑斯堤眼中的憤怒，俯著的頭更加貼近地面了。傑斯堤接著說[總而言之你和人類生活太久了，馬上停止那不像樣的宰相遊戲，專心在這大地要塞中找尋露克立斯那個人類埋藏的啟示。
艾米諾特因為無法照做，慌忙地回答[如果要讓其他人類無法公然地出現在這四周，宰相的爵位還是很有用的。等我做完幾項措施，裝出我是因壽命已盡，自然而死之後，我會儘快回到這裡完成您所吩咐的工作，還請您允准。]{np}雖然傑斯堤並不滿意他所說的話，但最後還是接受了艾米諾特的請求。[總之，你是第一個殺死拉伊瑪所選的人類守護者的人，看在這一點，我就答應你吧。]
得到允許的艾米諾特，立刻又提出了下一個請求。[還有一件事。[是什麼？]{np}普利米歐艾米諾特煩惱著自己的第二次請求要怎麼向魔王傑斯堤表示，然後他說，[大地要塞的鄰近都市路克索那還住著許多露克立斯追隨者。目前我的士兵們有在進行出入管制，但也不能一直這樣下去。
一直放任他們住在裡面會造成妨礙，將他們流放也會有問題。流放者中不乏有人接下了露克利斯偷偷留下的指示。但也不能因為這樣就將都市裡所有的人都關起來..]
[我不想再聽到人類社會那些煩人的事，那麼瑣碎的事不是我需要知道的。還是說說你的目的是什麼吧]
[請用傑斯堤你的能力讓整個都市滅絕吧。][這種事你應該也做得到吧?]
[我如果花時間做準備，收集珍貴材料，並執行完整的儀式，這樣當然是有可能的。然而讓一個都市裡的人類全數滅絕這種事，女神是絕對不會坐視不理的。畢竟還有協議不是嗎?]
[你為了成為貴族家族的一員，不也把那個家族的人殺了嗎?那時候女神都沒站出來了，這種事哪會有什麼問題?她們應該不會知道吧?]{np}艾米諾特回答。[嚴格來說，那時死掉的人，都是被傳染病害的。雖然散播傳染病的是我，但是被疾病害死的並不能算在我頭上，
再加上那地區的居民們得到了我很多的好處。重點是，那次的規模並不會像這次的這麼大。如果是這程度的規模，我認為女神會認為那是嚴重的違反協議。]{np}傑斯堤聽了此番話之後，陷入了沉思。她對人類一點興趣也沒有，但女神就不同了。對吉特涅策劃的事稍有了解的傑斯堤，在這個時刻，對大幅刺激女神之事十分謹慎。
畢竟在她們之間，還有著無法打破的協議。
稍有違反協議者，過去都以道歉或賠償收場，卻從來沒有故意要嚴重打破協議過。{np}不久後傑斯堤開口了。[我們這樣做吧，我會對那個都市降下石化的詛咒。如此一下，理論上來說，那裡的人們詛咒被解開時即使已經過了百年，也能復活，這樣就不是對他們趕盡殺絕了。
然後你就發揮你的長處，教唆人類的國王，將化為石像的人們移往其他地方。如此，我們雖然是妨礙了人類的生活，但並不是殺了他們。而這之後發生的事也都會是人類國王的命令導致，澤米娜要追查我們的理由就大幅減少了。]{np}
[真是卓見。如果加上這件事是因為女神對於反叛者露克立斯的行為感到憤怒而下的詛咒的傳聞，效果會更好的。]{np}[那種事情你自己看著辦。你要謹記，我們的目的是拉伊瑪的啟示，並不是為了殺害啟示守護者的人類。
吉特涅大人即使知道你們所有人都沒有任何成果也沒有為此處罰你們，你們要感到萬分的感謝，然後更加的努力。]語畢，傑斯堤就消失了。{np}普利米歐艾米諾特這才起了身。他從現在開始有很多事情要去做，但是那時他還不知道自己會在漆黑的地下要塞中，度過一段漫長的歲月且無法讓那件事情成功。
唯一如果讓他知道會感到欣慰的事情，是拉伊瑪得知自己所選的露克立斯遭受冤枉而死之後，真心的為此感到難過。無論如何，在女神與魔族之間的戰鬥中，這的確是魔族獲得的小小勝利。
