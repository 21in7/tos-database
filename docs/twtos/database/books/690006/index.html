利迪亞夏班和侏儒怪

Content

{s35}利迪亞夏班和侏儒怪{/} {np}王國各地流傳著有關利迪亞夏班的趣聞軼事。
甚至從歷史上的記錄來看，連可斷言利迪亞夏班不曾去過之處，也有其行經村莊時所發生的軼聞。
這種軼聞的內容很荒誕，比方說有龍出沒，或者出現性格不同於我們所知的魔族與利迪亞夏班行打賭之約等。
{np}儘管如此，該地的居民卻依然深信不疑，自己的故鄉出現在與利迪亞夏班有關的故事之中，這讓他們感到驕傲。
{np}相較於因為內戰而早死的露克立斯，或不常離開自己所建造的塔的弗羅里等人，利迪亞夏班確實曾在世界各地遊走冒險，所以很適合出現在那些軼聞之中。
從現在起所描述的故事也是那些傳說之一。
{np}也許故事本身不屬實，而且除了今日，在當時應該也是荒謬絕倫，但這種故事具有生命力並且流傳至今，此一事實就能說明其符合利迪亞夏班的性情。{np} 有一天，利迪亞夏班獨自離開某個村莊並走在夜道上。
利迪亞夏班看到有個旅人無力地癱坐在林間小路後便停下腳步。
利迪亞夏班出於好奇而開口問他。
[請問您怎麼會在深夜裡坐在森林中呢？]
跌坐在地的旅人在利迪亞夏班接連的催促下勉強回答。
旅人所回答的內容如下。
{np}據說旅人在幾天前經過這片森林時遇到名為侏儒怪的魔族。
旅人被捲進魔族的紛爭中，結果就和侏儒怪展開比試。
{np}雖然贏得勝利就能得到豐厚的獎賞，但與魔族的比試並不簡單，結果旅人便在比試中輸掉了。這下旅人只能唉聲嘆氣，等著把說好的獎賞拱手交給侏儒怪。
利迪亞夏班聽了原委後，心中萌生想會會那個名為侏儒怪的魔族的念頭。
他答應旅人會盡力幫忙。
過了一段時間，利迪亞夏班所等待的侏儒怪出現了。{np}利迪亞夏班向侏儒怪提議立刻無條件放了旅人並與自己比試。
不知是幸還是不幸，侏儒怪熟知利迪亞夏班的名聲，並且因為能和大名鼎鼎的利迪亞夏班比試而興奮不已，於是便接受了這個要求。
{np} 雖然現在侏儒怪之名不太為人所知，但當時這個惡魔因為比賽對手很強而一決勝負，最後因戰勝對方而聞名。
但牠總是對比賽規則做些微的改變。
利迪亞夏班回答說，
[嗯，我聽得出來你想跟我比射箭。可是你想怎麼做呢？]
{np}[比誰的箭術高沒有意思。所以我們這麼做吧，你決定靶子後射箭。那麼我就在那支箭射中靶子之前跳到那支箭上。]
侏儒怪說完後利迪亞夏班便問。
[你要跳到箭上？]
[沒錯。]
{np}侏儒怪回答後身體就開始變小了。
侏儒怪縮成拳頭般的大小後便接著說。
[怎樣？變得這麼小就能跳到你的箭上了吧？]
利迪亞夏班暫時沉思後說，
[這場比賽有問題。如果因為你碰到箭導致箭落地害我無法射中標靶，那不管你有沒有跳到箭上都是我輸啊？]
侏儒怪回答利迪亞夏班。
{np}[那這樣好了，我贏的條件就是跳到你的箭上，並且和箭一起飛到靶子上。]

[如果在那之前你從箭上摔下來呢？]
[那也算我輸。然後我們來比試三場。也就是說先取得3戰2勝的人就是贏家。]
{np}#VALUE!{np}利迪亞夏班搭上箭後把目標喊了出來。
[那邊最大的褐色樹木！]
說完的同時便放開弓弦，而箭則以驚人的速度破風而出。
{np}然而侏儒怪卻以輕盈的動作追上飛出的箭並騎了上去，當箭射中樹後，牠便配合那股震動在箭上跳舞並唱出不知明的歌。
利迪亞夏班再次搭上箭並大喊。
[我左側正前方松樹的所有松果！]
與之同時，利迪亞夏班便使出神乎其技的弓術。{np}好似炎熱夏天暴雨擊出的雷擊速度般的箭速，數十支箭，不，也許有數百支箭幾乎同時朝相同方向齊發而出。
{np}這次就連侏儒怪也無法從容不迫了。雖然用人類的肉眼根本無法辨別，但侏儒怪在飛箭劃破天際時，縱身跳到每一支箭上坐下後又跳到其他箭上，並且在最後一支箭射中松果前，乘著那支箭命中了標靶。
侏儒怪好像很累似地露出疲態對利迪亞夏班說。
{np}[總之，在最後一支箭射中標靶之前我已經騎上每一支箭，所以這場比賽是我贏了。]
利迪亞夏班回答說，
[你的確在所有箭射中標靶前跳到箭上，我同意你的說法。]
{np}侏儒怪聽了利迪亞夏班所說的話便露出邪惡的微笑說，
[那我們的比試是我贏囉。那麼有關剛才所說的獎勵…]
侏儒怪話還沒說完利迪亞夏班就搖著頭說，
[你該不會贏了兩次就不比最後一次了吧？ 我可不能只比兩場就認輸，而且我們明明約好比三次不是嗎？ {np}在完成3次比賽之前都還不算結束，我認為在比賽尚未結束前提及獎勵是沒有意義的…]
{np}#VALUE!{np}然而當牠騎在箭上朝向月亮飛去之前，馬上就明白這是一場無法結束的比試。
過了一段時間，箭矢因為力量耗盡而掉落地面，但利迪亞夏班射出後侏儒怪所乘坐的那支箭卻以靜止的狀態浮在半空中。{np}#VALUE!{np}根據傳說，侏儒怪還沒抵達月亮，並且使盡自身所擁有的魔力正往月亮飛去。
也許在牠的魔族壽命結束之前還回得來。
{np}然而如同我們所知，利迪亞夏班在很久以前就已進入永眠，雖然惡魔沒輸，但也永遠贏不了。
還有一件事讓身為後世的我們感到好奇，那就是利迪亞夏班與侏儒怪所提出的獎賞到底是什麼呢？
當然，我們還有人可以問，不，應該說是惡魔。但是我們不知道他什麼時候回來，看來那個疑問在短時間內應該很難解開。
