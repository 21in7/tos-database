蘇科姆的星之塔攻略紀錄1

Content

「若是要靠力量，應該要在塔完工之前執行才對」蘇科姆想著。
蘇科姆開始對夏班史達展開了攻擊。
第一階段是蘇科姆要集結所有能動員的魔族。
但是要讓位於人類世界的魔族一次大量的移動，並不是簡單的事，重點是還可能會遭受女神們的注意。{np}不幸中的大幸是，利迪亞夏班在世界到處流浪時，不斷的殺害了許多魔族，其中還包含了有名望的高階魔族。此舉可以被延伸為復仇戰，所以在與女神們的協定前，更顯自由。{np}不管是正當的一對一，還是因魔族有錯在先，在利迪亞夏班的正當防衛下被殺死的，這些都不重要，
要讓這麼多的魔族移動，就是件很困難的事情，更重要的是與利迪亞夏班和創造她的夏班史達團體的戰鬥，有可能會是一場持久戰，因此魔族就必須要不斷地派遣魔族來支援，同時還需要有值得信賴的部下。
這個任務就交給了名叫弗貝塔的高級魔族，
被稱之為魔君主並不是就能隨心所欲地動員魔界的魔族，就如小時候有次哈勃克說過的一樣，如所有的魔族都有完整的組織起來，沒有任何一個魔族是沒有所屬的話， 那應該會很好。{np}但是那種麻煩的事情，根本跟他們的天性不符合。
必要的時候聽從高階魔族的命令，有合意的高階魔族時，就直接進入他的麾下就可以了。創造主給了女神與他們充分的能力與自由。
所以那樣的組織化，跟他們的天性是完全不符合的。
況且，如果真的需要所有魔族的力量，也會由吉特涅大人親自召集所有人。{np}所以不管喜歡還是不喜歡，就算所有人都是在吉特涅的命令下去對付某人的，也能夠展現出他們集結的力量，
隨時都可以辦到的事情，現在也不需要這麼做，所以什麼所屬、組成組織，都是沒有效率且不方便的事情，
儘管如此，想起哈勃克的想法的人好像還是很多，
總之，蘇科姆在魔界募集參戰的魔族，並且動員智能低的魔物等等，花了不少時間準備，
另外最後在到達含星湖水時看到的，是完工的星之塔的模樣。
