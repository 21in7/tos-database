利迪亞夏班的箭尾槽

Content

{s35}
利迪亞夏班的箭尾槽{/}{np}根據傳聞，這是某天發生在星之塔的事。
某天，利迪亞夏班的門徒在星之塔中展開爭論。
據說多名門徒在訓練後閒聊途中出現這種話題。
[無論再好的弓，如果使用者能力拙劣就無法把威力發揮出來。因此重要的是自身的能耐。]
其他門徒反駁了某門徒的這種主張。
{np}[你忘了弓箭手的基本。何謂弓箭手？ 弓箭手就是使用弓這種工具的人。沒有弓的弓箭手能存在嗎？ 就算自身的武藝再高，如果用刀與槍施展武藝，那怎麼能成為弓箭手呢？ 總歸，弓箭手必須依賴工具。]
{np}於是有不同見解的門徒再次反駁。
[既然你說工具重要，那我們就針對這點來說吧。
所謂的弓的定義到底是甚麼？就像石弓這種比較不需要使用者能力的武器，以及會被使用者能力大幅左右的巫蠱師的武器，弓箭手能使用的武器也是很多元的。
巫蠱師的風箭是很難以弓去控制的，即使用再好的材料打造，沒經驗的人就是很難上手。]
{np}然而石弓或十字弓真的是誰都能使用的武器嗎？其實不然。
依照你的主張，必須有工具才會有弓箭手的存在，但這是錯的。{np}#VALUE!{np}如果沒有風和空氣，射出的箭與子彈就無法命中目標，如此一來就不會有神射手的存在，而名弓又有什麼意義呢？ 最後，對弓箭手而言，最重要的是去感覺風的流動，並讓自己的意志與風結合。]
{np}此後門生的爭論也不斷持續下去，並演繹出更多樣有趣的主張與理論。
最後門生也開始好奇他們的師父─利迪亞夏班是怎麼想的。
所以門生都去找利迪亞夏班，並把至今的對話告訴他等待利迪亞夏班解答。
{np}據說當門生來找利迪亞夏班時，她正好在用箭頭的尖刃剝栗子吃，並且危險地跨坐在星之塔的欄杆上。
{np} 利迪亞夏班繼續吃著栗子並傾聽門徒們反覆論述的主張，他暫時安靜地不動聲色，接著就用手指撥弄自己吃完後堆積下來的栗子殼，並從中拿起某樣東西如此說道。
[這是什麼？]
一名反應快、眼力好的門徒最先回答。
[那是栗子的內皮。]
{np}那確實是栗子內皮，因此其他門徒也點點頭等待師父開口說話。
於是利迪亞夏班便繼續說下去。
[不對。這是我的箭尾。]
{np}聽說利迪亞夏班說完就把栗子的內皮夾在手指之間，並用它包住時常置於身邊的箭的尾端，將之當成箭尾槽般使用並把箭射向天際。
據說利迪亞夏班用奇怪的方法消除門徒們的爭論，並從欄杆上縱身而下留下不知所措的門徒們離去。
