利迪亞夏班的轉折點

Content

{s35}
利迪亞夏班的轉折點{/}{np}王室中流傳著一個記錄：國王卡杜梅爾問利迪亞夏班:[就算是妳應該也射不中藏在樹或石頭後面的靶子吧？]此時利迪亞夏班簡明扼要地應了一聲，同時用響亮的弓弦聲代替回答並命中標靶。
然而有關於此，王室的正式記錄與流傳於世間的傳說中尚有未提及的部分。那就是卡杜梅爾國王說出那番話的目的，其實不單純是為了測試利迪亞夏班的箭術。
{np}更重要的是，當時利迪亞夏班所射中的標靶是什麼東西，不，正確說來，她射中的是什麼人才是重點。
王室的記錄與民間的傳說談到此軼事時，兩者都未提到這是什麼時候發生的事。
{np}只是把焦點放在利迪亞夏班強大的箭術上，將其當成英雄故事來說。
之所以如此是有原因的。
因為這軼事是發生在內戰時期的最後階段。與部分流傳的傳說不同之處是，露克立斯與卡杜梅爾國王的士兵戰鬥時，並不是在大地要塞的最後一層戰死。流傳露克立斯是因為某個原因，不想讓外人進入大地要塞最後一層的中心。
{np}所以露克立斯是在大地要塞淪陷之前逃出要塞，而卡杜梅爾國王則調動士兵追擊他。
追擊到最後，露克立斯被逼到懸崖邊。{np}那個懸崖上正好有個巨大的岩石，露克立斯背對那塊岩石，以正面朝向懸崖的狀態對付國王的士兵。
當露克立斯背對那塊岩石時，追擊中的國王弓箭手就攻擊不到露克立斯了。
如果是其他場所，就能藉由繞道從對面與左右包圍猛烈射箭，但這類攻擊對懸崖陣型是行不通的。
{np}因為沒有弓箭手能越過懸崖在虛空中進行攻擊，因此卡杜梅爾國王簡直是一個勁的把士兵逼向懸崖邊。
{np}#VALUE!{np}在這種情況下，卡杜梅爾國王想出的詭計就是他向利迪亞夏班提出的問題。
當然，利迪亞夏班並沒有因為卡杜梅爾國王的挑釁而上當，也沒為了展現自己的能力而用驚人的曲射殺害岩石後的露克立斯。
此處又有另一個隱情。
{np}說到那個隱情，身為後代的我們也只能猜測那是利迪亞夏班的師父的願望，而親手結束某人生命的心情一定是既痛苦又複雜。
總之因為這個事件，世人皆知迪亞夏班能射穿被岩石擋住的標靶，但她當時複雜心情的轉折點是無從推度的。
