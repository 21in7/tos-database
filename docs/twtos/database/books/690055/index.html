魯希德 萬托斯芬的 備忘錄

Content

王國建國曆679年的某一天
-
最近有了能和泰斯拉大人見面談話的機會。泰斯拉大人說，在過去歲月你雕刻的各個雕像，會證明自己的歷史並喚醒回憶，獲得迎接未來的力量。並且也對我問道，我是否也有類似的東西。
我告訴他沒有，因此泰斯拉大人建議我最好也建造一個。我和泰斯拉大人道別後，經過仔細思考，決定接受世界上唯一比我年長的長者忠告。
這就是我之所以會留下備忘錄的原因。
{np}
王國建國曆700年的某一天
-
琳卡烏達斯在凱利高原建造教堂時向我提出了諮詢。但是我過去也只看過幾眼，怎麼可能記得已經消失許久的聖殿結構呢?
琳卡烏達斯似乎一直對我提出時間探索，但是我打算告訴他，若是要建造守護秘密的建築，與其照著過去的樣貌，不如建造一個全新的更能守住秘密。
不過，因為他喜歡梅伊班大教堂，所以應該不會有什麼大變化才是。總之，聽說7年以後會完工，到時後再去替他祝賀就好了。
{np}
王國建國曆749年的某一天
-
統治王國40年的索謝爾國王的健康似乎不是很好。雖然我覺得以目前的狀態，應該還能撐個2、3年。但是這似乎會是極限。
在將近750年歲月的王國歷史中，被評為最賢明的君主統治之下生活的歲月，似乎會成為美好的回憶。
{np}
王國建國曆872年的某一天
-
已經將近半百的奈克爾3世的的統治，似乎也已經達到極限了。因為沒有看過紀錄，所以不是很清楚。不過像這樣在位這麼久的長壽國王，確實是前無古人，後無來者。
雖然不如索謝爾1世國王的時期，不過記得當時盛事的人民寥寥無幾的現在來說，要說目前是王國史上最好的太平盛世，實在是一點都不為過。
{np}
王國建國曆876年的某一天
-
呵呵，奧文狄班拒絕了這麼多次，最終還是沒能拒絕國王的遺志，接受了王室魔法師的位置…看來以後會展開相當有趣的時代。
{np}
王國建國曆879年的某一天
-
奧文來找上我問道到底要過著隱遁生活到什麼時候。自己被王室絆住就算了，居然還想拖我下水…我覺得有點唐突。{np}
總之，在他苦苦哀求之下，
最後決定向王室捐贈部分我的研究成果。
我看，第一份捐贈物大概就是備忘錄了。
雖然不知道會是什麼時候…{np}
王國建國曆884年的某一天
-
我知道在我背後，那些看似中階魔法師的孩子們，
在談論現今最強的魔法師是不是我。
他們認為能與我一較高下的代表性人物，就是奧文狄班。他是世人經常在說的魔法代表，也是王室的首席魔法師。他的修練時間比我少了將近一百年，但是卻擁有能與我匹敵的實力。{np}
但是若真的賭上性命，
兩人展開決鬥時，我可以確信他至今還不是我的對手。除非遭到殺害，或是自己選擇死亡。考慮到以魔法延緩時間永生的我們，我相信總有一天他一定會超越我。
但是根據我的推估，這個時間還需要另一個一百年。
而另一個競爭者死靈法師來說，她就算再多修練一百年，也沒辦法超越我的。
{np}
王國建國曆890年的某一天
-
我打算在註記一些在這份紀錄上。雖然有提到過魔法師間的決鬥，不過不管是奧文狄班、我，或是任何人，在目前都無法與亞佳伊拉弗羅里比較。這不只是表示她具有強大的魔力。{np}
她的氣勢方常強大，視野也非常寬廣。
對我們後輩來說，只能對她仰望而已。
世人難道不懂嗎?確實有過能察覺弗羅里之大器的代表性事件的事。{np}
她為了守護女神的請求
引起了讓其他女神接續她之後的前所未聞的大事件。她為了粉碎對魔君主來說非常遠大，卻對人類太過危險的慾望與計畫，訂下契約降伏了他。
最後，讓他瀕臨崩解，並以不是以女神超越般的身分，而是以一介人類的身分踏入了幻想圖書館。透過這件事，徹體粉碎了吉特涅的計畫。{np}
為了她善良的目的，讓女神降臨到了
魔法師之塔，並利用魔君主動搖了魔神的計畫。光是這點就能足以證明她是偉大的魔法師。{np}
雖然現在大家還不知道，不過總有一天
所有人都會知道這件事的。不過，就算沒有發生這件事，亞佳伊拉弗羅里的名聲
已經非常高到無法再更高了。

Description
