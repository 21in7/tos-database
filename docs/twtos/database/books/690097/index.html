克里戴拉歐泰羅因為不喜歡他，所以都沒有跟他說話，在漫長路途和長時間的沉默下，率先投降的普里昂先開口說了。
{np}[老實說，我總是對妳感嘆不已。大師所援助的二十九名少年少女中，能夠達到大師標準的人非常稀少。但妳不只達到了那個標準，還取得了那以上的成就。

普里昂按照規定敲打了在他前面的門，開門後，看見除了粗糙的桌子和椅子之外沒有其他任何東西的房間裡坐著的武僧大師。

[坐吧。]

{np}[突然有一個不相干的人要坐在那個位置上..難道競技場其他的管理者們，還有現在鬥牛士大師的弟子們會乖乖待著看嗎?]
[那個我和普里昂會解決的，還有雖然妳不太清楚，但鬥牛士大師身邊的人其實都是很喜歡妳的，果然女人的外貿出眾的話，也是有很多的好處。
{np}所以只要妳用實力將他們壓制，讓他們了解到在鬥牛士的技術上面也不輸於他們的話，應該就不會有問題了。而且我認為在明年的這個時候，妳的技術應該就會超越一生都在修練的那些人了。]
克里戴拉雖然沒有再說任何話，但也還沒答應，感覺正在思考著什麼一樣。
{np}武僧大師看看之後問道。
[感覺妳還有煩惱的事情卻沒有說出來，應該是跟我有關的事情吧。]
[我只是暫時在思考大師之間禁止私自戰鬥的王國法而已。]
{np}武僧大師因為那句話而輕聲笑了出來。
[哈哈，是這樣嗎?我知道妳在擔心什麼了。但鬥牛士是競技場的大師，而我也是個喜歡站在競技場的大師，對妳來說機會是很足夠的。
{np}但如果是想殺死我的話，妳應該要先償還妳所欠下的債務吧?我很確信妳再償還債務之前不會把我怎麼樣的，雖然妳的實力很厲害，但我是因為欣賞妳的價值觀，所以才收留妳到最後的。]

{np}[妳這樣就不符合成本了，如果我們只有提供妳食衣住行的話，也許是足夠的。但我們為了將妳打造成擁有最頂尖武藝實力的人而提供了所有的劍法和訓練，而且不只是妳小時候所欠的債務，連妳家門所積欠債務都幫妳還清了，還有..]

{np}..前略..
王國的歷史上，首次出現競技場和劍鬥士的時間，大約是在900年前的梅基耶爾國王時代。
之後競技場和劍鬥士漸漸的發展起來，分類出各種的基礎職業，一直延續到現在。
{np}但在大約300年前，普利內爾Freenel國王的在位期時，引入了王國外部的風俗習慣後，在競技場原有的劍鬥士中誕生了和各種基礎職業完全不同與競技場相關的鬥牛士職業。
{np}更加詳細敘述的話，本來是以我們世界的動物、魔族或是同樣身為人類的劍鬥士為對象來戰鬥和表演的競技場，將會多一個以魔獸為對象來戰鬥的競技表演。
{np}魔獸雖然是同屬於魔界的產物，但和魔族明顯不同，是一種沒智慧或智慧很低的野獸，不存在於這個世界上，可以說是魔界特有的動物。
對付魔獸的這種事情，而且還是在競技場中，與之前我們世界的動物進行競技表演的劍鬥士們經驗相比，需要更多的技術和情報。
{np}這時，王國競技場的相關人員偶然從王國外部社會裡擁有對付魔獸經驗的部族社會中獲得了情報和技術，他參考了自己所接觸到的情報和技術後，創造出了一個全新對付魔獸的方式。
{np}這完全是一種全新的型態和方式，這代表誕生了新的競技場工作者。
到晚年為止，他將自己所掌握的王國外部技術再加上自己所創的技術，把這一切都整理好，最終得到了大師位置的公認，鞏固了鬥牛士這個新職業的誕生。
{np}初代鬥牛士所創立的鬥牛士技術和方式，包含了實際戰鬥所需要的內容和為了給觀眾看的藝術與綜藝，可以說是一個綜合格鬥藝術。
{np}然後經過了300年的時間，鬥牛士的技術果然也隨著時間而產生了變化。
其中要選出最特別的?變化的話，就是鬥牛士的技術不止可以對付魔獸，再一對一的戰鬥中也很強大。
{np}鬥牛士在進行過程中，雖然也有得到輔助者們的幫助，但最終還是需要獨自一人面對魔界的魔獸，面對如此強大的魔獸，經過了三個世紀，因為這樣，鬥牛士的技術也跟著往特殊化的方向去發展了。
{np}因此現代的鬥牛士跟過去專門以魔界魔獸來進行表演的前輩鬥牛士不同，反而是以一對一的戰鬥專家而得到了更多的名聲。
