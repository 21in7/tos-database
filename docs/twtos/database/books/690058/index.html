現代大師總覽
{np}1. 索勒夏河頓(Sorsha Hutton)

河頓家門的元老生於拉耶爾2世時代,那時拉耶爾幫助庫素梅爾國王贏得王位爭奪立下大功勞, 因此獲得了貴族的頭銜與領弟.
歷經數個世紀河頓發展出家門獨有秘傳劍術,那個時代身為貴族必須精通學術, 懂得經營領土以及武術等等,要是連這些基本常識都沒有,會被驅逐到領地外,被迫閉門修練劍術.
{np}索勒夏河頓在小小的年記就表現出家門史上難得的特有天賦,且自小就自願擔任擊劍手大師的職位.
家門史上不見得每一位擊劍手大師都是屬於河頓家門出身,不過河頓家門的人自願擔任大師的的話,一定至少具備了無法被取代的大師實力,所以才會年紀輕輕實力就能被家門內外的人給認同.
{np}2. 羅勒塔 寧帕斯(Loreta Nimbus)

羅勒塔 寧帕斯生於多米爾國王統治時代末期,普利內爾國王在位期間,地位就已經提升到了大師等級.
超越了古代的大師,她最大的成就就是發現了吸收生命力享有長壽的方法.
當然,這個方法第一個受惠的就是他本人,後來成了羅勒塔安基納斯的一份子之後, 死靈法師大師也跟其他幾位大師一樣反而不需要挑選繼承人了.
{np}3. 吉那 葛林(Gina Green)

葛林家門在整個王國國內是擁有最多領弟的貴族.
吉那 葛林從小就在家門領地內活動長大,自小經歷過的許多事情對於他決定未來要成維德魯伊以及晉升成大師很有幫助.
{np}她雖然繼承了葛林家門多數的領地,德魯伊大師指對一些特定的事情有興趣,管理領土這種事情實在做不來,所以就把大部分的土地給賣了,委託給凱都拉商團後,她成了讓寬恕者大師羨慕的有錢人 .
也不能說是投資或是很有遠見的舉動,神樹之日讓全國的土地與農場瞬間成了廢墟,她先前的選擇反而為她自己帶來許多財富.
{np}當然吉那本身並不是那麼愛錢的,所以才能幫助那些神樹之日的受害者以及損失慘重的家門經濟上的援助.
{np}4. 雷歐那羅德 奎東(Leonard Quicktongue)

雷歐那羅德 奎東不是工兵大師的本名.
從10幾歲開始就是個聲名遠播的盜賊,曾經因偷竊軍隊物品 導致身為軍職的父親代替兒子背負罪名讓家中名聲留下汙點.
因為兒子的事情一句話都無法辯解的父親,聽說到死前都沒見過工兵大師一面.
{np}奎東日日反省自己的過錯,雖然成了軍人,最後當上工兵將校立下許多功勞,但是他反覆提出面會要求父親身為上位者,聽說只回答若是命令可以答應面會要求.
不知道是不是因為那件事情的關係,工兵大師不知不覺繼續默默做著善事.
