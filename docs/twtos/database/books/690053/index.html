在王國歷史開始被記載的
許久之前，這塊土地上就流傳著
一些關於神秘又驚奇的
故事。
這本書記載著
在那些故事之中，
最為人知曉的三個傳說
故事。{np}在無數的傳說之中，
選定往後要記錄場所
的原因，是因為

民眾對寶物與黃金等財富
的關心，相當濃厚，
甚至成為女神信仰
的基石。

現在在三個故事之中，
我就來為你簡單介紹
巴羅昆士黃金礦脈的傳說吧。{np}第3章：巴羅昆士黃金礦脈
-
聽說王國外某座隱密的山中有傳說中的
黃金礦脈。
根據傳說，先發現礦脈的
竟然不是人類
而是魔族，是有位勇士把魔族
擊潰之後，把礦脈搶過來的。{np}那位勇士只是怕魔族把黃金拿去
對人類做壞事
才把礦脈搶走的，作為不貪圖財富
、富有榮譽心的勇士把礦脈的
正確位置當作祕密，也說絕對不會
把礦脈裡的
黃金做為己用。{np}歲月流逝，勇士也逐漸年老
壽命走到盡頭，也沒留下任何後代
傳說中的金礦
也變成沒有主人的狀態，如果有人
發現的話就可以
成為最富有的人了。{np}不過這個傳說
有個疑點。
不管是在王國的歷史中或是其他任何的
紀錄裡，都沒有找到任何從魔族手中搶過礦山，名為
巴羅昆士
的英雄。{np}因此也有很多人認為
巴羅昆士根本是不存在的人
所以巴羅昆士的黃金礦脈
也不過是個傳說罷了。
傳說裡對於礦山的名字
為什麼叫巴羅昆士
也沒有特別說明。
即使這樣，還是有人
深深相信這個傳說，
也從未停止探索。
