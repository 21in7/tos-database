Content

利迪亞夏班是位名弓箭手。
在起風的日子裡，她的箭一樣也可以射中蟲子的觸角。
這女孩站在最強劍士露克立斯前方。
她停止呼吸，緩緩地拉起箭絃。
利迪亞夏班不久前也這樣注視著曾經是戀人現在卻為敵人的露克立斯。
{np}冷風刺骨的某個冬天。
卡杜梅爾王國的滿橫暴政無視道德規範，
持續強迫勞役與忍受飢餓的居民們，腐敗的領主，無能且殘暴的國王…
露克立斯再也無法忍受下去了。
起義。
路克索那所有人都相信露克立斯這次能夠贏得光榮的勝利歸來。
{np}四季流轉，春天再度來臨。露克立斯沒想到氣勢日漸衰弱。
露克立斯收到了和王軍勾結的盜賊居住的村莊情報。
王軍到底是敗壞到何種地步…
露克立斯下定決心帶領部下們前往攻擊村莊。
{np}才準備要面對盜賊群後，卻因為莫名的緊張感讓緊緊抓住劍的手感到麻木。
突然間如風聲被劈開的聲音般，幾名同伴胸口飛濺出一道又一道的鮮血，一個個暈倒了。
同時，露克立斯和他的部下們大喊幾聲後往村莊方向跑去。
就這樣，毫無預警地戰鬥正式開始。
