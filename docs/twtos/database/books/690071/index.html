米爾堤斯的故事

Content

雖然受到傑斯堤的命令而讓各方魔君主們花了很長的時間尋找著拉伊瑪的蹤跡，但沒有任何一個魔君主成功。
追捕永遠可以預先得知未來的女神，絕對不是一件容易的事情，但是最近終於對追捕拉伊瑪行蹤的事情有了一點進展。{np}艾米諾特認為，那是因為他殺了拉伊瑪所選的露克立斯，導致拉伊瑪受到了刺激而疏忽了，但看在米爾堤斯的眼裡，那根本就是不可能的事情。{np}另一方面，拉伊瑪製造啟示非常吃力，顯示她的能力或許正逐漸地衰退。
總之在米爾堤斯看來，他認為自魔君主追蹤拉伊瑪以來，現在他追她比任何人，比過去任何時候都更加積極。
夏奧雷伊森林中發生了一件事，讓他相信可以抓得到她。雖然那之後暫時讓她逃了，但他認為要再找出來並不難。因為米爾堤斯擁有其他魔君主沒有的偵查方法。
果不其然，他在人類建造的監獄裡發現了與一個男人待在一起的拉伊瑪，但在這裡，他還是以微小的差距讓對方跑了。
然而自追蹤拉伊瑪以來，從來沒有一個魔君主像他一樣，能與拉伊瑪待在同個空間，直接看得見她。{np}米爾堤斯對於這點感到非常的樂觀，並確信下次發現她時，絕對不會再讓她逃走。
根據米爾堤斯短暫看到的模樣來看，拉伊瑪是個看起來很虛弱的人類少女模樣，基本上狀態看起來也不是很正常，看起來並沒有能力對付米爾堤斯。{np}所以在水晶礦山裡發現拉伊瑪的瞬間，米爾堤斯深怕拉伊瑪又再次逃跑，不斷的檢查著可能會逃亡的地方。
那項檢查已包含了可以瞬間移動的可能性。
拉伊瑪最好有只要任何人在這個礦山中使用瞬間移動，礦山就會瞬間崩塌的準備。
拉伊瑪知道自己已經無法從那裡逃出去之後，讓米爾堤斯更加感到開心。{np}可能是因為這樣，走到拉伊瑪面前之前，一直都沒有留意到她身邊的那個人類男子。
站在拉伊瑪面前的米爾堤斯開始了這場對話。[這件事從開始到現在，能直接與你碰面的，似乎都只有我一個人。所有的魔君主連妳的影子也沒看見，追著追著就放棄了，不過從現在開始，漫長的追蹤要在我這裡落幕了。]
拉伊瑪雖然什麼話也沒說，站在拉伊瑪身旁的人類男子，坎寧安開口了。[你就沒想過我們是故意在此處等你的嗎?]{np}米爾堤斯說道。[說話的語氣真是不恭敬。總之不管你跟拉伊瑪是基於什麼原因停留在這裡，結果都會是一樣的。況且，你們人類難道不認為沒有可以介入我們的資格嗎？]{np}米爾堤斯不再理會坎寧安，再度對拉伊瑪開口說道。[根據吉特涅大人嚴格的命令，遇到妳的時候，我們必須對妳進行一次勸說。
但是如果妳到最後都不願意接受我們的勸說時，我們也收到了要不擇手段取得啟示的命令。所以，希望妳在我們還對妳客氣的時候乖乖合作。]{np}然而拉伊瑪什麼話也沒說，坎寧安便繼續說下去。[我們之所以決定在此處等你，當然也是因為很難再逃下去。不曉得你聽說了沒有，水晶礦山是被開採了數百年的礦山。]
[你想說什麼?]
米爾堤斯一說完，坎寧安便立刻做出回答。[那是很古老的礦山了，挖不出水晶的礦區，就等於是荒廢了。因此就算這裡真的倒塌，誰也不會知道這裡曾經有人，更不會有人想來開挖。]
[所以呢?]{np}坎寧安回答了這個問題。[因此若是我將你封印，並使這個礦區坍塌，你會因為封印的魔力而永遠被關在裡面，礦夫也不可能沒事來挖開這裡把你弄出去。而且根據拉伊瑪所言，倘若你失敗了，你所謂的上級，也就是魔王，他不可能來救你。也就是說你已經玩完了。]
說出這句話的同時，坎寧安拿出了一個圓形的物體。仔細一看像是紡線用的紡車，整體看來長得像一個輪子。{np}這是題外話了，這又被稱為未來四大聖具，王都發生植物災難，世界樹暴走的時候，當時是聖具第一次被使用，它縮小了災難的規模，幫助穩定局勢。
雖然日後在這不遠處，還會發生初代聖殿十字軍大師琳卡烏達斯留下的另一個聖具擊退米爾堤斯上級傑斯堤的事件。總之米爾堤斯身為魔族，受到這個能擊退魔王的聖具攻擊，算是第一個遭受聖具攻擊的案例。{np}當米爾堤斯想要做些什麼的時候，她終於發現了自己周圍的時間流動是異常的。
那意味著像激流般流動的時間漸漸的變得平靜，最後變成了像湖水一樣靜止的狀態。
如果時間被停住了，應該可以用空間移動來逃出那裡，但是她明白這條路已經由自己親手阻斷，米爾堤斯動彈不得。{np}雖然米爾堤斯認為應該要在那湖泊之水變成不會因風而再起漣漪的冰凍狀態之前，自己要在紡車輪型聖具的影響力下逃出，但他已感覺到身體就連精神都無法控制了。
就在米爾堤斯完全被束縛住，到處都看不到冰卻全都像被冰凍後，坎寧安說[這聖具的威力還真是厲害。]
拉伊瑪說[那也就是我在它身旁的原因，要是人類獨自使用，下次就不會出現像現在這樣的威力了。]
坎寧安說[沒關係，這魔君主看來得在這裡待上一段時間，快點離開吧，就由我來收尾。]
[希望你不要做你打算做的事情。]
[不，你不是知道我辦不到嗎？我要弄倒礦山，這樣才能阻止其他魔族或是偶然來到這裡的人類引發問題 ，尤其..]{np}坎寧安頓了頓又繼續說。[最重要的是，告訴我們這個聖具的目前所有人如果死了，就會消失到別的地方去的也是拉伊瑪大人。所以為了未來，這個聖具不能留在這裡。所以讓這裡崩塌，讓我死在這裡的話，就可以同時完成兩件事情。
死掉的露克立斯也一定會贊同我的作法。否則，他為了女神而死掉，實在是太可憐了不是嗎？]{np}拉伊瑪在聽到露克立斯的名字後，又再次感到悲傷，最後同意了坎寧安所說的話。拉伊瑪祝福坎寧安的犧牲後離開了那裡。不久後現在的記憶又會再次因長久的歲月而被遺忘的這件事情，再次讓這瘦弱女神的心感到悲痛。
不知道是慶幸還是不幸，拉伊瑪在聽到坎寧安引發的爆音之前，離開了水晶礦山。
