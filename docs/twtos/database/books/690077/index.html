蘇科姆的星之塔攻略紀錄3

Content

查看湖水的結果，並沒有獲得任何的成果。如果那只是放滿女神神聖力的聖水，那麼還會有解決的方法，但這看起來並不那麼簡單。只是憑感覺認為是混合了魔法、煉金術以及當代神學，但並沒有找到明確的解決方法。雖然不斷的研究，總有一天會得到答案，但現在是束手無策的。{np}在部下的犧牲搜查之下，獲得的成果是查出了有個名叫聖水盤的物體存在，從那裡產出的液體混合在湖水中，這種水只會對魔族有害，對人類或其他動植物是不會造成任何影響的。
如將那水盤裝置全部破壞，說不定湖水的濃度就會降低變成一般的湖水，但散佈在各處的設施就算夏班史達團員在沒有任何妨礙之下，也是很難一一去破壞的，因雖然湖水只到腳踝處，但卻沒有不弄濕腳而能接近的方法。{np} 蘇科姆雖然有很多有能力的部下，但是夏班史達團員個個都是名射手。除非一次派出壓倒性的數量，否則想要接近水盤設施是根本不可能的事情，即使靠近了也會被弓箭射死。
而在偵查湖水上空的飛行型魔族，也被星之塔發射的大型弩弓攻擊而死或受到重傷。{np}所以下一次是試圖利用船進行水上攻擊，這個名叫含星湖水的地方是在擁有有不知道深度的深淵上，用魔法在湖上建立塔後，就算要靠近也是需要用魔法來製造出通道的，這個通道非常地寬，甚至可以說不是路，而是廣場或叫做其他的名稱，
總之，這些全都是用魔法所組成的石材，是靠魔法的力量漂浮在深淵湖水上的，這裡是廣闊的湖水有一半左右都是由浮游魔法石所組成的水域，而其他地區就只是湖水，所以他打算划船到沒有浮游魔法石的水域上去攻擊塔。{np}但是船隻這種東西又不是需要就能馬上生出的東西。
因此船隻的準備花費了很多的時間。弗貝塔在魔界中收集著船隻，並且拉攏了可以操控船隻的魔族，但為數還是遠遠的不足，最後還是秘密的與普利米歐艾米諾特聯繫上。他在人類的世界中，累積了非常龐大的資金，他獲得了那些資源，購買了人類的船隻，並飛向了含星湖水。{np}在搭上船後，首先湖水至少就沒有傷害到部下們了，星之塔那邊看起來沒有正規的水軍或是水上兵力，
在緩緩接近時，也都沒有展開任何的攻擊，
但誰知道就在到達看似是星之塔的水上某處時，星之塔那裡就瘋狂地發射了投射武器，
部下們就算沒直接被擊中，也因濺起的水受傷，就算沒有被擊中要害，船隻被擊中也開始進水了，
要是身體碰到水，是會受傷的，但也沒辦法把水舀出去。{np}但是蘇科姆並沒有愚笨到沒有預想到敵人會有這種強力的遠距離型投射武器。{np}蘇科姆的目的並不是星之塔的投射武器有多少，而是分析是否有可以對水面上的敵人發動攻擊的手段。
即使祭出了龐大的犧牲，也要想辦法接近到星之塔的附近，但是隨著距離越拉越近，除了投射武器之外，只有夏班史達團員們發射的箭矢增加，除此之外並沒有其他的攻擊手段。蘇科姆確認了這一點之後，下令讓大家撤退。{np}另一方面，又讓飛行型怪物總動員，來個水上與路上同時的進攻，但是即便如此，還是無法接近星之塔的周圍。不過，他並不是為了確認可以預期到的事實，還特地製作了船隻，讓部下一一犧牲的。
