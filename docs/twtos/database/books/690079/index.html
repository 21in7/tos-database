蘇科姆的星之塔攻略紀錄5

Content

下一個要促進的方法，就是雇用人類的傭兵。
往好處去想，只要有了魔族與人類傭兵的聯合軍，在戰術的運用上的確是好事，但是即便是花錢請來的傭兵，沒有人不會在乎與魔族並肩作戰的感覺。
所以簽約時，還必須偽裝成人類的模樣才能順利進行。{np}但是即便提出再高的金額，當傭兵們聽說攻擊對象是一般民眾時，大家還是搖搖頭直接拒絕。
即使能克服這個階段，但只要聽到是要與含星湖水的利迪亞夏班敵對時，每個人都認為這是發瘋的行為，並且巴不得快離開。{np}當中還有人對王國舉報，導致王國的官吏開始對想要徵求傭兵攻擊星之塔的事情開始展開了調查。
也因如此，更加沒有人願意加入傭兵的行列。{np}而往返著魔界與人類世界，在一旁輔佐著他的弗貝塔，將這件事報告給上層，蘇科姆立刻被魔王傑斯堤叫回去臭罵了一頓。
傑斯堤極度討厭魔族依賴人類的行為，被傑斯堤得知這件事之後，除了成了傑斯堤的眼中釘，也成了其他魔王討厭的對象，讓他的立場變得越來越難堪。{np} 蘇科姆雖然對弗貝塔提出了警告，但對弗貝塔來說，這件事嚴重到必須向上級報告，所以也無法太過於苛責他。
面對蘇科姆的進攻，利迪亞夏班與他所帶領的夏班史達，到目前為止都還沒有對外提出救援的請求，甚至對於這裡發生的戰況也沒有傳的太遠，這對他來說算是不幸中的大幸，不過這也代表了夏班對於這場戰鬥有著相當的自信心，也做足了相關的準備。{np}其實在這個時間點蘇科姆雖然不知道，但是之後幾個世紀，再沒有外部支援之下塔要能頂得住，利迪亞夏班從一開始希望有支援，不管是王國的官方支援或是民間人士的援助，但是其實自己很清楚不能期待支援。
甚至利迪亞夏班也依慮這一點，她所經歷過像是卡杜梅爾國王一樣的作者以後再出現，甚至和王國軍和魔族聯合，打算奪取塔和他所留下的啟示的狀況必須也要承受得住。蘇科姆在之後的數百年歲月間，確切地感受到此事實。
