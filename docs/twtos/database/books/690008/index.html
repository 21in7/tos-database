卡杜梅爾時代的盔甲

卡杜梅爾時代的盔甲{/} {np}

巴道斯 溫特史普恩 著{np}

內戰結束後過了30年，我突然有個學術上的問題，於是我針對它研究了10年，而其研究近來終於有了成果。
我認為這個研究成果小至滿足我個人的學術好奇心，大至有助於提高我所屬的溫特史普恩家門的各項研究。
{np}言歸正傳，我花了很長的時間搜索大地要塞與石化都市，與露克立斯及其追隨者有關之處也已徹底勘查過，在這之後我所做出的結論如下。
另外，我認為這與杜梅爾國王駕崩後由他人所編纂的內戰時期的記錄相符，而且完全沒有矛盾之處。
總而言之，在內戰期間所發生的戰事中，叛軍與國王軍之間的一般武裝是沒有差異的。{np}我認為雙方的武器與盔甲及軍需品在質量上並沒有任何差異。
雖然卡杜梅爾國王的兵力比較多，但站在只須保護大地要塞及其周遭都市(現稱石化都市)的叛軍的立場想，{np}我的結論是攻擊者的龐大兵力並沒有對其造成太大的負擔。
就如自古以來所有的兵書所言，攻方與守方的比例小至3對1、大至10對1，都是對守方有利。再加上如果把大地要塞這種由露克立斯所建造的要塞作為守城之地，即使擁有20倍的兵力，就攻城者的處境來說並不算多，這是軍事學者們一般的說法。
{np}不過有件事不能忽視，那就是卡杜梅爾國王的軍隊是由單純的正規軍組成，其兵力所擁有的不只是優越性。
比方說，無論是當時或現在，王國的戰爭與千年前的戰爭，在戰術方面存在很大的差異。{np}雖然軍事學上有很大的發展，但使其呈現出最大差異的關鍵，就是與過去無法比擬的魔法學與鍊金術的發展。
根據魔法師與隨軍祭司介入戰爭程度的多寡，現場指揮官必須留意以前沒顧慮到的許多要素。{np}而露克立斯與卡杜梅爾國王的軍隊作戰時，最應費心的關鍵其實就是這部分。
{np}如前所述，內戰時期雙方的武裝與其他裝備及兵力的差距雖是無意義的，但根據推測，投入屬於正規軍的卡杜梅爾國王軍隊的魔法師與祭司的數量相當龐大。
{np}雖然露克立斯的師父─梅伊班的影響力在不知不覺間讓許多祭司推諉從軍，但隨軍祭司的數量依舊可觀。由於魔法師與露克立斯沒有任何情緣，所以有大量的魔法師自由加入戰局。{np}一般來說，在軍事裝備與士兵數量旗鼓相當的狀況下，魔法師大舉加入戰局應該會造成很大的影響，但實際上內戰時期的戰局並沒有如此發展，這是眾所皆知的事。
{np}針對此現象，從過去就有各種分析。有人批判卡杜梅爾國王愚昧、無能，無法善加運用魔法師，也有人說露克立斯卓越的勇猛克服了一切。
{np}雖然這類見解並不是一切，但本研究的結果欲對其補充一件事。那就是露克立斯與其軍隊所使用的盔甲；尤其是有關露克立斯的盔甲的真相。{np}接著我們要觀察那段時間的研究結果，在戰鬥中，魔法之所以對露克立斯與其追隨者無效，最大的原因是因為他們的盔甲對魔法具有極高的耐受性與抗性，此即為本研究主張的核心。
{np}換句話說，本研究算是主張露克立斯與其部隊的盔甲是以強大的魔力合金製成，因此可無視魔法師的攻擊魔法。
乍聽之下，也許這類見解聽起來很荒唐。{np}對魔法帶有如此強大的抗性的魔法合金存在一說並不常見，假如這種魔法合金真的存在，只有身為指揮官的露克立斯擁有的話還說得過去，但如果能供給所有部下穿那就太誇張了。
很有可能有人會提出這類反駁，而且這說法也非常妥當。{np}即使是寫這本書的作者，當其他人提出這類主張，這可說是以合理的方式來思考時，會馬上浮現在腦海中的駁論。
況且為了支持這種主張，必須補充另一個與我們所熟知的物理現象背道而馳的荒誕假設。
{np}如果不是筆者親自研究所得到的結論，我自己也很難認同這種假設。
那個牽強的假設就是，特定的魔法合金即使被低溫凍結也不會毀損。一般人認為鋼鐵之類的金屬非常堅硬，但其實它對溫度十分脆弱。無論再剛硬的金屬一旦被猛然結凍，就算是受到小小的衝擊也會產生裂痕或被擊碎。
{np}但我認為露克立斯所採用的魔法金屬，是自然界中唯一帶有違背此現象的特性的魔法合金。
{np} 我走訪大地要塞和其他與露克立斯有關連的地方後所下的結論是，露克立斯把這種來歷不明的少量魔法合金，用在自己的追隨者的武裝上，而這些少量的魔法合金就足以防備想攻擊多數對象的國王軍魔法師的廣範圍魔法。
{np} 如果把焦點轉移到露克立斯身上，對他發動猛攻的魔法失效；尤其是最後一刻國王與魔法師已然放棄，而士兵們的犧牲不斷增加，導致最後國王不得不仰賴利迪亞夏班的箭。參考這個記錄，此假設便更具說服力。
{np}若以發現的其他幾個線索來推論，當時露克立斯所穿的盔甲，的確是大量使用上述的魔法合金製作而成。

考量到當時寒冰法師最積極出手朝他猛攻，我認為這盔甲與其材料─魔法合金對寒氣的抵抗力是不容想像的。{np}因此本筆者暫時 '將此魔法合金命名為'寒鐵
