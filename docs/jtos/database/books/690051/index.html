王国南部キレン地方にはアレンビック洞窟という場所にまつわる話が伝わります。

過去にとある凄腕の錬金術師が
ありったけの力を振り絞り、金銀財宝はもちろん、貴重な遺物を作り出し、これを自慢するために、たまに人々を
外部から招待するという話です。{np}{@st66b} アレンビック洞窟がある場所を
知る人はいないと言います。

しかし、隣の村に住む年よりたちは彼らの子孫に、消えた人々は錬金術師に招かれ、彼に会いにいき、そこから出たくないほど裕福な人生と長寿を享受しているとのことです。
従って、この村で原因不明の理由で突然人がいなくなった場合、彼らの不在を悲しむよりは自分もいつか招かれる日が来るだろうと期待しながら
失踪者たちの幸せを祈る風習が
定着していきました。
